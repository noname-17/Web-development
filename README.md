# 📖 About The Project

This project is a **responsive multi-page website** created using **HTML5 and CSS3**.  
It demonstrates modern front-end design techniques including responsive layouts, structured navigation, UI components, and clean styling.

The website includes several pages such as **Home, About, Blog, Services, and Contact**.

The project is deployed using **GitHub Pages**.

---

# ✨ Features

✔ Responsive design  
✔ Modern UI layout  
✔ Navigation bar with hover effects  
✔ Hero section with gradient background  
✔ Blog cards layout  
✔ Course / service section  
✔ Contact page  
✔ Footer with social icons  
✔ Smooth animations using AOS
